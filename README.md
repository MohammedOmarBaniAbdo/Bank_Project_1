# Client Management System (C++)

A console-based client management system built using C++.

## Description
This project is a simple client management system that allows the user to manage client records using a file-based system.

The program stores client data in a text file and provides a menu to perform different operations.

## Features
- View all clients in a formatted table
- Add new clients
- Search for a client by account number
- Update client information
- Delete a client
- Data is saved in a file (MyFile.txt)

## Technologies Used
- C++
- File Handling (fstream)
- STL (vector, string)

## How It Works
- Each client record is stored in a single line in a file
- Data is separated using a delimiter (#//#)
- The program reads and writes data from/to the file

## How to Run

Compile the program:

g++ main.cpp -o app

Run the program:

./app
